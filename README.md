## Overview

A website for an Asian restaurant, consisting of three main pages: Home, Menu and About Us. The Home page features a description about the restaurant and a call-to-action button. The Menu page showcases three dishes. The About Us page provides information about the restaurant's.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/aderitomarrime/restaurant-page.git
```

2. Navigate to the project 
```sh
cd restaurant-page
```
3. Install dependencies (requires Node.js installed)
```sh
npm install
```

4. Run the development server
```sh
npx webpack serve
```

5. Open your browser at http://localhost:8080


## How to use

1. Open the website in your browser.
2. View the home section.
3. Use the navigation links to change the pages.
